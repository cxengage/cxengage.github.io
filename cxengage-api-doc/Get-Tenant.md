# Tenant Resources

```
  GET /tenants/:tid
```

## Description

Retrieves tenant info

```

```

**Return**

```json
    "id": "tenant1",
    "name": "Tenant Number One"
}

```

```
curl -X GET http://192.168.0.28:8182/tenants/tenant1 \
```
